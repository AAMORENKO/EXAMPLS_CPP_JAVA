Текст шкільної олімпіади (ПРОГРАМУВАННЯ)
(папка для зберігання: Прізвище_Клас\*.*)

1. (55 балів) папка UNIQUE, файл UNIQUE.CPP
Перевірити, чи всі елементи масиву Q [N] цілих двозначних чисел є унікальними (N> 99).
 
2. (65 балів) папка SUMSUB, файл SUMSUB.CPP
Максимальна сума підмасивів. Нехай дано масив C[N] чисел; задача полягає в тому, щоб обчислити максимальну суму підмасивів, тобто найбільшу можливу суму послідовних елементів. У масиві зустрічаються елементи з від'ємними значеннями з відрізку [-9; 9].

3. (75 балів) папка QUEEN, файл QUEEN.CPP
Обчислити кількість способів розставити двох ферзів на  дошці N × N, так щоб ніякі два не били один одного. Так, на малюнку показано вісім можливих рішень для N = 3, (0<N<11).

4. (45 балів) папка PASCAL, файл PASCAL.CPP
Вивести на екран дерево Паскаля для числа N, (N> 8).

5. (95 балів) папка SUBGRIDS, файл SUBGRIDS.CPP
Дана сітка N×N з чорними (1) і білими (0) клітинами, обчислити кількість підсіток, у яких усі кутові клітини чорні. На малюнку показані дві такі підсітки.
