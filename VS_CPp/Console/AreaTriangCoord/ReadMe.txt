1) А(0;0) В(4;0) С(0;3)

        Обчислюємо площу трикутника за координатами його вершин


        Введiть координати точки А -> x1 y1 -> 0 0

        Введiть координати точки B -> x2 y2 -> 4 0

        Введiть координати точки C -> x3 y3 -> 0 3


         Знайдено довжини стор?н трикутника:

        a=    4,00
        b=    5,00
        c=    3,00

        Площа трикутника дорiвнює S=   6,000 кв.од.

        Для продолжения нажмите любую клавишу . . .
