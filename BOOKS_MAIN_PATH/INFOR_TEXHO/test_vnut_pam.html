Внутренняя память компьютера


  1. К внутренней памяти не относятся:

    ОЗУ
    ПЗУ
    СМОS
    жесткий диск

  2. Свойством ОЗУ является:

    энергонезависимость
    перезапись информации
    энергозависимость
    долговременное хранение информации

  3. Свойсвом ПЗУ является:

    только чтение информации
    энергозависимость
    перезапись информации
    кратковременное хранение информации

  4. Быстродействие ОЗУ зависит от ...

    быстроты нажатия на клавиши
    частоты операций записи или считывания информации из ячеек памяти
    напряжения питания
    типа монитора

  5. Наименьший элемент оперативной памяти - это ...

    ячейка
    регистр
    байт
    файл

  6. ОЗУ размещается

    в процессоре
    на жестком диске
    на магистрале
    на материнской плате

  7. Объем ОЗУ измеряется

    в байтах
    в пикселях
    в ячейках
    в МГц

  8. Совокупность программ для тестирования компьютера и загрузки операционной системы - это...

    CMOS RAM
    Flash Memory
    Bios
    ROM

  9. Кэш-память - это ...

    внешняя память компютера
    внутренняя память компьютера, используемая при обмене данными между процессором и ОЗУ
    постоянная память компьютера
    нет верного ответа

  10. Работа с пользовательскими программами начинается после того, как они будут считаны из внешней памяти в...

    ПЗУ
    ВЗУ
    ОЗУ
    CMOS RAM
